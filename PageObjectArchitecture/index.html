Architecture Page Object

Le code présenté ici suit l'architecture Page Object, une approche largement reconnue pour développer des tests d'automatisation maintenables et évolutifs.
Les fonctionnalités et les éléments du DOM spécifiques à chaque page sont encapsulés dans des classes dédiées, à savoir LoginPage et HomePage.
