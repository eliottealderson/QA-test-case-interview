Gestion des dépendances

Nous utilisons npm comme gestionnaire de paquets pour gérer les dépendances de notre projet. Pour installer les dépendances, exécutez la commande suivante dans le répertoire de votre projet :

npm install

Vous pouvez ensuite exécuter vos tests en utilisant la commande : npm test.
